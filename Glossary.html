Finding Prerequisites Information
Term Definition/Explanation
Banner ID Refers to a number which is assigned to every student as a way of identifying individuals with distinct numbers.
Class Code Refers to a student's classification i.e. FR - freshman, SO - sophomore, JR - junior, SR - senior
Class Title The descriptive name of a course which is formal title, while also specifying a specific section of the class. Ex: Software Engineering
CRN Course Registration Number - a distinct code that refers to a specific section of a certain class within a certain semester. These are not unique from one semester to the next, so the term code will also be considered when the program searches for a CRN.
Hours Calculated based on classes that the student has taken.
Major Code Refers to a specific student's declared major.
Schedule Calculated based on the times specified for sections, including days of the week and time of day.
Sections A way of differentiating between multiple instances of one course within the same semester. One course may be offered several times within one semester, and it is important to be able to specify which group of students is being referred to.
Section Number The number that refers to a specific section. This can be represented in several ways: a number, i.e. 1, simply refers to a standard course section; H and a number, i.e. H1, refers to a section of an Honors course; T and a number, i.e. T1, refers to a section of a course that is for transfers students.
Term Code An numeric code which specifies the year and season (i.e. fall, spring, summer) in order to refer to a distinct semester.
Subject Code An abbreviation which refers to the subject being studied. Entering this information gives the user access to either a specific CRN or a list of specific courses contained in that major that the user could then enter to narrow their search.
